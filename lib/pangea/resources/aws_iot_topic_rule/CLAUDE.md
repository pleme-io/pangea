# AWS IoT Topic Rule Resource - Claude Documentation

Manages IoT topic rules with intelligent SQL analysis, action validation, and message routing optimization.

## Key Features
- **SQL Query Validation**: Validates IoT SQL syntax and complexity
- **Action Type Analysis**: Categorizes and validates rule actions
- **Error Handling**: Configures error action handling
- **Performance Analysis**: Assesses SQL query complexity and performance impact

Enterprise-grade message routing with comprehensive validation and optimization recommendations.
